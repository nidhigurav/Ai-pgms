📘 AI Search Algorithms & Problem Solving Programs

This repository contains implementations of classic Artificial Intelligence search algorithms and problem-solving techniques using Python.

---

🚀 Programs Included

1. Water Jug Problem (DFS)  
File: pgm1.py  

- Solves the Water Jug problem using Depth First Search (DFS)  
- Takes input for jug capacities and target amount  
- Displays the sequence of steps to reach the solution  

Concepts Used:  
State Space Search  
DFS Algorithm  
Backtracking  

---

2. Missionaries and Cannibals (Best First Search)  
File: pgm2.py  

- Solves the Missionaries and Cannibals problem using Best First Search  
- Ensures safe states where missionaries are not outnumbered  

Concepts Used:  
Heuristic Search  
State Validation  
Priority Queue  

---

3. A* Search Algorithm  
File: pgm3.py  

- Implements the A* algorithm to find the optimal path in a graph  
- Uses both actual cost (g) and heuristic (h)  

Concepts Used:  
A* Algorithm  
Heuristic Functions  
Graph Traversal  

---

4. AO* Algorithm  
File: pgm4.py  

- Implements the AO* algorithm for solving AND-OR graphs  
- Determines the optimal solution path  

Concepts Used:  
AND-OR Graphs  
Heuristic Optimization  
Recursive Search  

---
